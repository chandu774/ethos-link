# 🚀 Synapse – AI Powered Community Chat App

Synapse is a full-stack, real-time social platform that combines **communities, chat, and AI** into a single application.  
It is designed to feel familiar like WhatsApp/Discord, while introducing smarter community discovery and AI-assisted conversations.

🔗 **Live App:** https://ethos-link.vercel.app/
🔗 **GitHub Repo:** https://github.com/chandu774/synapse

---

## ✨ Key Features

### 💬 Chat System
- Real-time private & group chat
- Message replies and reactions
- Typing indicators & online status
- Read receipts (sent / delivered / seen)

### 🌐 Communities
- Explore communities with real data
- Hybrid model:
  - Platform-created communities
  - User-requested communities (admin approval)
- Community roles:
  - Admin
  - Moderator
  - Member
- Pinned messages & community rules

### 👤 User Profiles
- Unique usernames (@handle)
- Custom display name
- Profile photo upload (local storage → cloud)
- Role & badge system

### 🧠 AI Features
- AI-powered message summaries
- Ask AI inside communities
- AI highlights important discussions
- AI messages visually separated from user chat

### 🛡️ Moderation & Safety
- Report users and messages
- Block users
- Role-based permissions
- Admin approval workflow for communities

### 🔔 Engagement
- Per-community mute
- Presence & last seen
- XP, levels & leaderboards (gamification)
- Polls and community events

## 🧑‍💻 Tech Stack

**Frontend**
- React (Vite)
- Tailwind CSS / CSS
- JavaScript

**Backend**
- Supabase
  - Authentication
  - PostgreSQL Database
  - Realtime subscriptions
  - Storage (images)

**Deployment**
- Vercel

---

## 🏗️ Architecture Highlights

- Realtime chat powered by Supabase Realtime
- Secure authentication with Row Level Security (RLS)
- Scalable community & role-based permission model
- Modular and maintainable React component structure
- Production-ready build & deployment
